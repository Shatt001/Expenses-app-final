# Description

Expenses tracker app. App allows to create, edit, delete expenses. App also able to filter and sort expensis by some criteria. <br />
<br/>
App contains 4 pages:<br/>
Login page (only google authentication service available)
Home page (List all not deleted expenses)<br/>
Add expense (add new expense)<br/>
Edit expense (edit or remove expense)<br/>

# Available Scripts

In the project directory, you can run:

### `npm run dev-server`

Runs the app in the development mode.<br />
Open [http://localhost:8083](http://localhost:8083) to view it in the browser.<br/>
The page will reload if you make edits.<br />

### `npm test`
Unit test is imlemented for some parts of application.<br />

### `npm run build`

Builds the app for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.
